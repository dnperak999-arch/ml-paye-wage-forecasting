# Machine Learning Coursework – PAYE Wage Forecasting

This project explores UK PAYE RTI data to forecast sector-level average wages.

The workflow includes:
- Exploratory data analysis
- Feature engineering with lagged variables
- Baseline vs ML model comparison
- Model evaluation using standard error metrics

**Tech stack:** Python, pandas, scikit-learn  
**Status:** University coursework (2025–2026)
